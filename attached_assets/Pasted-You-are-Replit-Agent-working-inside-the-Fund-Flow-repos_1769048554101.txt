========================================================
A) PII REDACTION (BEFORE MODEL CALL)
====================================

Goal: Ensure **no raw PII is ever sent to OpenAI**.

1. Create module:
   `server/lib/privacy/piiRedactor.ts`

Expose:

```ts
redactText(
  input: string,
  mode: "off" | "standard" | "strict"
): {
  redactedText: string,
  findings: {
    type: "email" | "phone" | "iban" | "bank" | "card" | "id" | "token" | "address",
    count: number,
    examples: string[],
    confidence: "high" | "medium" | "low"
  }[]
}
```

2. Minimum redaction rules (regex + heuristics):

* Emails → [REDACTED_EMAIL]
* Phone numbers → [REDACTED_PHONE]
* Credit cards (Luhn) → [REDACTED_CARD]
* IBANs → [REDACTED_IBAN]
* Bank a/c numbers → [REDACTED_BANK]
* API keys / tokens (sk-, rk-, JWT-like) → [REDACTED_TOKEN]
* IDs / passports → [REDACTED_ID]
* Addresses → redact ONLY in strict mode

3. Implement:

```ts
redactObject(obj: any): any
```

(recursively redacts all string fields)

4. Integrate into **every OpenAI call path**:

* User message
* Extracted PDF text
* Web snippets
* Agent outputs before re-prompting

5. Add Copilot UI control:

* Privacy dropdown: Off / Standard (default) / Strict
* Backend receives: `{ privacy: { pii_mode } }`
