   - growth + retention stability

   - confidence penalty

SIMULATION ENGINE (Monte Carlo):
- Monthly time steps; horizon 24 months.
- baseline from truth scan values.
- knobs: pricing_change_pct, growth_uplift_pct, burn_reduction_pct, hiring plan, fundraise, collections_improvement_days (optional), gross_margin_delta_pct (optional).
- stochastic:
  - growth_rate_t ~ Normal(mean=baseline_growth + uplift, sigma configurable)
  - margin varies with sigma
- cashflow:
  cash[t+1] = cash[t] + (rev[t]*gm - opex - payroll - other_costs) + optional_fundraise
- outputs:
  - runway distribution (months to cash<=0)
  - survival probabilities at 6/12/18/24
  - p10/p50/p90 bands for revenue, burn, cash
  - survival curve points

DECISION ENGINE:
- MUST use deterministic candidate action library below (do not randomize).
- Select 8–12 actions deterministically based on truth metrics:
  - If confidence < 60: only conservative runway + fundraise + pricing (3/5) + burn cuts; exclude marketing push & aggressive growth.
  - If runway_p50 < 6: include opex20/30, freeze hiring, fundraise M1/M3, bridge+burn10, combo burn20+raise.
  - If concentration > 50%: deprioritize marketing push; prefer pricing/GM/burn actions.
  - If retention 12m low: deprioritize marketing push; include engineering/platform and GM improvements.
- For each candidate action, generate scenario deltas, run simulation (or cache identical), compute scores:
  survival_score = P(survive 18m)
  runway_score = clamp(runway_p50/24, 0..1)
  growth_score = clamp(revenue18m_median/baseline_revenue18m,0..2)/2
  efficiency_score = map burn_multiple improvement to 0..1
  risk_penalty deterministic:
    - if confidence<60 and bucket=growth => +0.15
    - if runway<9 and adds hires => +0.10
    - if burn_reduction_pct>25 => +0.05
    - if pricing_change_pct>=8 => +0.05 unless confidence>75 and retention strong
  total = 0.45*survival + 0.20*runway + 0.15*efficiency + 0.20*growth - risk_penalty
- Return top 3 actions with:
  title, rationale, expected impact vs baseline (delta survival_18m, delta runway_p50), risks, key assumption, next validation scenario delta.
